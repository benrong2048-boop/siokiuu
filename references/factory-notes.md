# 工厂技术资料：内部理解与沟通边界

> 来源：工厂考察会议纪要。该文件用于提高内部判断质量；它不是自动对客话术来源。与网站或说明书不一致时，以公开资料和负责人确认结果为准。

## 负载与功率

| 负载类型 | 示例 | 工厂资料的倍率说明 |
|---|---|---|
| 纯阻性 | 白炽灯、电加热器 | 约 1 倍 |
| 感性 | 电机、风扇、搅拌机 | 约 1.5–3 倍 |
| 特殊非线性 | 定频空调、焊机 | 约 2–5 倍以上 |
| 电动工具 | 电锯、切割机 | 约 3 倍 |
| 活塞式空压机 | 空压机 | 不低于 3 倍 |
| 螺杆式空压机 | 空压机 | 不低于 5 倍；当前产品线无法带动 |

定频空调在启动/制冷阶段可能有较高压缩机功率；不能仅按稳定运行功率选机。对客户只说明需要核对设备的运行功率与启动功率，不保证适配。

## 保护与故障

- 过载时，先减少/断开负载，确认启动浪涌和设备接入顺序；不得把过载直接判断为发电机故障。
- 工厂资料提到启动困难可能涉及化油器浮子等内部问题，但公开 FAQ 禁止客户自行内部维修。因此客服只给外部检查和技术售后升级路径。

## 并机与场景

- 工厂建议同型号并机；不同型号“可行但不推荐”。
- 工厂含 90% 并机效率计算示例，与网站约 80% 表述不同；不得对客使用该示例作为统一承诺。
- B2C 主力场景为家庭/露营；工地/工业场景占比较小。大型工业设备、焊机、螺杆空压机需谨慎处理。

## 维护与待确认

- 工厂资料与网站在部分维护间隔、并机效率、型号/接口表述上存在差异。
- 面向客户时优先按对应型号说明书和当前公开网站；遇到差异登记“待确认”。
